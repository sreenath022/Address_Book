**A Java Project**

 Developed a comprehensive Address Book application to e ciently manage and organize contact
 information. The application provides features for adding, editing, deleting, and searching for contacts,
 ensuring user-friendly interaction and robust data management

**Functionality**

Print all available contacts

Add new contact

Search contact based on name

Search contact based on phone number

Edit contact based on name

Edit contact based on phone number

Delete contact based on name

Delete contact based on phone number
